# uuid-cl

## Description
Simple little program to use for generating uuids on the 
command line. 

## Building

```go build```


## Usage
```uuid <optional args>```


#### Optional Arguments

-n=# Number to generate (default = 1) 

-d=true|false Print dashes in uuids (default = true)

-s=string Separator character for multiples (default = newline)
